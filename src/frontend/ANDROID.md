# Android Build Guide

This guide explains how to build a downloadable Android APK from the E.L.B.E.R.R. horror game.

## Prerequisites

Before building the Android app, ensure you have the following installed:

1. **Node.js** (v16 or higher) and npm/pnpm
2. **Java Development Kit (JDK)** 17 or higher
   - Download from: https://adoptium.net/
   - Set `JAVA_HOME` environment variable
3. **Android Studio** (latest version)
   - Download from: https://developer.android.com/studio
   - During installation, ensure Android SDK is installed
4. **Android SDK Command Line Tools**
   - Install via Android Studio: Tools → SDK Manager → SDK Tools
   - Check "Android SDK Command-line Tools"
5. **Gradle** (usually bundled with Android Studio)
